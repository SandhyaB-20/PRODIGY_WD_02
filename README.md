# Stopwatch Web Application

This is a simple Stopwatch Web Application built using HTML, CSS, and JavaScript. It allows users to start, pause, and reset the timer easily.

## Features
- Start the stopwatch
- Pause the stopwatch
- Reset the stopwatch
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was developed as Task 1 for my internship at Prodigy InfoTech to practice JavaScript functionality and DOM manipulation.

## How to Use
1. Open the index.html file in a browser.
2. Click the Start button to begin timing.
3. Use Pause and Reset buttons as needed.
